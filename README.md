# r/place

A toy version of [Reddit r/place](https://www.reddit.com/r/place/), a
collaborative real-time canvas.

## Quickstart

- Start a local Redis server on port 6379.
- Start the server by `go run .`
- Open `localhost:8080` in the browser.
